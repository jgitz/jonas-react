- in the APP COMPONENT.... use IS-LOADING BOOLEAN STATE 

- TOGGLE THE IS-LOADING inside the USE-EFFECT 

- create the LOADER COMPONENT to display loading text 

- set CONDITION to display MOVIE LIST based on IS-LOADING STATE'S VALUE
